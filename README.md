# Bomber-Man

# 概要

学校の発表会に向けて初めて作ったゲームです。

二人プレイの対戦ゲームです。

プレイヤー二人が十字に爆発する爆弾で相手を倒すのが目標です。

# 操作方法

プレイヤー1はWASDで移動 Spaceで爆弾設置。

プレイヤー2は↑←↓→で移動 Enterで爆弾設置。

ESCを押すことでゲームを閉じることができる。
